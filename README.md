## How to Run

1. Download or clone this folder.
2. Open any `.html` file in a browser (Chrome, Edge, or Firefox).
3. No installation or build step required.

**To run logical JavaScript questions:** open terminal in the project folder and run:

```bash
node JavaScript/logical_question.js
```

**Recommended starting point:** open `Final_Project/index.html` to view the portfolio.
